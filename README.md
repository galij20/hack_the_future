# hack_the_future
project for hack_the_future hackathon 

Required Tools:
-npm
-tailwindcss

After git cloning:
1.cd server 
  npm install express mongoose dotenv cors plaid axios
2.cd client 
  npm install express mongoose dotenv cors plaid axios

Running the Application
1. Start the Backend

cd server

node server.js

2. Start the Frontend

cd client

npm start
